# Paradise Nursery - e-plantShopping

Welcome to **e-plantShopping**, an interactive React and Redux Toolkit e-commerce application for purchasing indoor plants.

## Project Overview
**e-plantShopping** (Paradise Nursery) allows users to browse categorized houseplants (air-purifying, aromatic, medicinal, etc.), view detailed plant information, and manage a dynamic shopping cart in real time. 

## Key Features
- **Landing Page:** Interactive welcoming page to get started.
- **Product Catalog:** Categorized plant cards with single-click "Add to Cart" functionality.
- **Shopping Cart Management:** Dynamic updates for quantity, subtotals, and total cost powered by Redux state management.
- **Responsive UI:** Clean, mobile-friendly interface built with modern React.
